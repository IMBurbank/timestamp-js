A Timestamp Microservice

Give me a try.

Pass a time as a parameter to the url of this microservice.

Microservice will return a date json string with unix and natural dates for any valid query. Values of null will be returned for all non unix or natural date queries.


Example usage:

'https://timestamp-js.glitch.me/July%2031,%202017' or

'https://timestamp-js.glitch.me/July 31, 2017' or

'https://timestamp-js.glitch.me/1501459200000'

Example output:

{"unix":1501459200000,"natural":"July 31, 2017"}
